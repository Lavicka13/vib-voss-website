# V.I.B. Voß Immobilien Beratung — Projekt-Workspace

Kunden-Workspace für die Website-Erstellung von Edith Voß / V.I.B. Voß Immobilien Beratung (Ilvesheim).

## Struktur

| Pfad | Inhalt |
|------|--------|
| `brand/` | Logo-SVGs, Porträt, Design Tokens (`design-tokens.json`) |
| `briefing/` | Aufbereitete Briefing-MDs aus Ediths Original-Materialien |
| `docs/superpowers/specs/` | Design Spec (Stand: 2026-05-21) |
| `docs/superpowers/plans/` | Implementation Plan (4 Wellen, ~30 Tasks) |
| `website/` | Next.js 16 Marketing-Site |

## Quick Start

```bash
cd website
npm install
npm run dev
# → http://localhost:3000
```
