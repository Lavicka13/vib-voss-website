## 5. DORN Immobilien — Suchformular für Immobiliensuchende

*Dateien: `20260520_110910.jpg` + `20260520_110227.jpg` — mit Anmerkungen von Edith*

URL: dorn-living-solutions.de/immobiliensuche

**Edith-Annotationen:**
- „Standort" → **„Wunschorte"** umbenennen
- „Wohnfläche", „Grundstücksgröße" → **„ca."** ergänzen (toleranter)
- „Budget" → **„Limitpreis"** umbenennen
- „Personen im Haushalt" → **gestrichen**
- „Haustiere vorhanden?" → **gestrichen**
- „Falls ja, welche?" → **gestrichen**
- „Barrierefreiheit / Aufzug" → **gestrichen**
- „Smart Home / Energieeffiziente Bauweise" → **gestrichen**
- „Lage & Infrastruktur" → **„Lage:"** umbenannt
- „Öffentliche Verkehrsanbindung / Schulen & Kitas / Einkaufsmöglichkeiten / Innenstadtlage / Arbeitsplatz in der Nähe" → **alle gestrichen außer „Ruhige Wohngegend bevorzugt"**
- „Maximale Nebenkosten" → **gestrichen**
- „Finanzierung gesichert?" → **gestrichen**
- „Wann möchten Sie umziehen?" → **gestrichen** (oder „sofort" → ok)
- „Wie dürfen wir Sie am besten kontaktieren?" → vereinfacht zu **„Telefon"**
- „An wen dürfen wir die Vorschläge schicken?" → mit „k.A." markiert / vermutlich überflüssig
- Pflicht-Häkchen DSGVO → ok

**Erkenntnis:** Edith will ein **deutlich schlankeres Suchformular** als DORN. Ihre Klientel (Premium-Käufer) füllt keine 20-Felder-Formulare aus. Empfehlung:

```
Schlankes Suchformular (V.I.B.-Version)
- Kauf / Miete
- Art der Immobilie (Villa, MFH, EFH, Wohnung, Apartment)
- Wunschorte (Freitext)
- Wohnfläche ca. m²
- Grundstücksgröße ca. m² (falls relevant)
- Limitpreis €
- Weitere Wünsche (Freitext, groß)
- Lage: nur „Ruhige Wohngegend bevorzugt" + freie Beschreibung
- Name, E-Mail, Telefon
- DSGVO-Checkbox
- → Senden
```

## 6. DORN Immobilien — Exposé-Stil (zur visuellen Inspiration)

*Dateien: `Objekte/*.pdf` (5 PDFs)*

Die Exposés von DORN Immobilien dienen als **visuelles Vorbild für Objektdarstellung**:

- Großes Hero-Foto oben
- Saubere Headline-Hierarchie: Firmenname klein → **Objekttitel in Türkisblau** → Foto
- Vier Kern-Daten direkt unter dem Foto: **Preis · Zimmer · Schlafzimmer · Wohnfläche**
- Strukturierter Datenblock: Ort, Wohnfläche, Nutzfläche, Grundstücksfläche, Anzahl Zimmer/Schlaf/Bad, Stellplätze, Provisionshinweis, Baujahr, Modernisierung, Zustand, Energieausweis
- Lange Beschreibungstexte mit ruhigen Absätzen
- Ausstattung als Icon-Liste mit „Yes/Ja"-Status

**Erkenntnis:**
- Edith möchte vermutlich ein **ähnlich ruhiges, hochwertiges Objekt-Layout**
- Türkisblau als Akzent passt NICHT zum V.I.B.-Logo (rosa/schwarz) — hier Edith-Logo-Farben verwenden
- Datenstruktur kann übernommen werden
- Hinweis auf **„nach Absprache"** bei Preisen erlaubt — Edith arbeitet diskret
